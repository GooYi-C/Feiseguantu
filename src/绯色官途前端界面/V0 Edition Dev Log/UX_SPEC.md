# 绯色官途 · 视觉与交互规范

## 1. 主题与基调
- 官方肃正系：深炭黑 #0f1117 / 卡片底 #161922 / 金色点缀 #d8a657 / 边框 #242838。
- 绯色互动系：暗红渐变 #c41e3a → #ff4d6d，用于绯色关系、警示、CTA。
- 成功/信息/警示/高危：#4ac18e / #7aa2f7 / #e0c36c / #ff6b6b。
- 背景尽量无纯色遮挡，使用磨砂渐变 + 轻阴影；保证无横向滚动。

## 2. 字体与排版
- 中文正文：`"Noto Sans SC", "Microsoft YaHei", sans-serif`；数字标题可用 `"Inter", sans-serif`。
- 标题层级：H1 20-22px / H2 18px / 卡片标题 16px / 正文 13-14px；行高 1.5。
- 间距：卡片内左右 16px，上下 12px；分区间距 16-20px；列表行 10-12px。

## 3. 布局与响应
- 主栅格：单栏滚动，卡片两列（≥1200px）；中屏单列，窄屏全宽 + 汉堡抽屉导航。
- 高度：使用 `aspect-ratio` 控制卡片比例（如 16/9、4/3），避免 vh。
- 导航：顶部标题 + 刷新按钮；左侧抽屉（宽屏常驻，窄屏汉堡展开）；面包屑显示当前层级。

## 4. 关键模式
- 卡片：圆角 10px，1px 边框 + 8px 内阴影，标题区带细分割线；右上角可放状态 Tag/按钮。
- 标签/徽章：圆角 999px，按语义上色；风险/危机使用暖色，机遇使用冷色。
- 手风琴：用于履历/项目/把柄/人情债分组；默认展开最近项。
- 抽屉/侧栏：人物详情、编辑表单、导入导出面板使用右侧 420-520px 抽屉。
- 覆盖层：遮罩透明度 0.55，允许点击关闭（除危险操作对话框）。
- 敏感遮罩：绯色/把柄字段默认模糊或遮罩，点击“解锁”后展示，支持 10s 自动重新遮罩。
- 关系网：SVG/@pixi/react 实现；节点圆形，尺寸随权重（靠山/宿敌更大），边样式与颜色按关系类型：靠山(金实线)、绯色(绯红渐变)、竞争/宿敌(锯齿/虚线红橙)、家属(柔和灰蓝)、普通人脉(银灰)。悬停高亮一阶邻居，其余降噪；右上角显示“总人数 / 当前显示人数”。
- 分类筛选：侧栏勾选“官场关系/绯色关系/竞争关系/靠山关系/家庭关系”后仅高亮包含对应子模块字段的节点；可组合多选。
- 绯色互动页：主色用暗红渐变，卡片带玻璃态；头像/封面可本地上传（不写入 stat_data），显示本地缓存状态与替换/清除按钮；卡片内含关系阶段进度条、危险度色阶、情绪标签、近期事件。
- 全量变量编辑页：顶层搜索框（字段名/内容）、“仅显示已填写”开关、“展开/折叠全部”控制；分区卡 + 手风琴，长 record/数组使用分页或虚拟列表；字段右侧提供“编辑”图标，点击行内切换编辑态。

## 5. 动效与反馈
- 加载：居中旋转 + “正在加载”文案；局部骨架屏用于长列表。
- 过渡：路由淡入淡出 200ms；卡片悬停轻微上浮；折叠/展开 180-220ms 高度动画。
- 刷新/写入：按钮旋转态 + 成功/失败 toast；写入失败保留表单数据。
- 风险/机会变化：delta 提示（▲下降/上升）+ 颜色过渡。
- 关系网交互：拖拽/缩放加速度阻尼；节点点击弹出抽屉时保留焦点；双击画布回到 {{user}} 中心。
- 全量变量编辑：进入编辑态平滑展开输入控件；保存/撤销有轻量过渡；分页切换保持滚动位置；局部保存成功后绿色短提示，失败红色提示并保留输入。

## 6. 表单与校验
- 分段表单：基础信息、关系模块、风险与通联、备注；长表单使用 Tab/手风琴分区。
- 校验：zod 预校验 + 软提示；必填字段高亮；提交前二次确认敏感操作。
- 输入控件：Tag 选择（多选）、数值滑条（0-100 能力/危险度）、日期/时间文本校验（HH:MM）、长文本 textarea 自适应高度。
- 批量导入导出：JSON 文本区 + 粘贴提示；导入前试算校验并提示差异。
- 本地插图上传：使用文件选择/拖拽，读取为 DataURL 或 Blob URL，存储于 localforage/indexedDB/localStorage，Pinia 仅存引用。提供“预览、替换、删除、清空缓存”操作；上传前限制尺寸与大小（如 2-4MB 内）并提示。
- 全量变量控件映射：枚举→下拉；0-100→滑条+数字框；文本→单行输入；长文本→可折叠 textarea；record→行表格（键名输入 + 值表单，含增删复制）；数组→可排序行（上移/下移）；布尔→切换开关；时间/日期→掩码输入。

## 7. 搜索与筛选
- 顶部搜索框 + 条件筛选（派系、角色标签、危险度、好感/信任区间、状态）；组合条件显示为可删除的 Chip。
- 快捷过滤：只看“高危”“机遇”“待办”“绯色对象”“靠山/竞争”等预设。
- 关系网筛选：关系类型多选、显示“只看有绯色关系”/“只看有靠山关系”等快捷开关；可按好感/信任阈值过滤边。
- 全量变量搜索：字段名/值模糊搜索，结果高亮并自动展开所在分区；可切换“仅显示已填写”隐藏默认空值。

## 8. 可视化指引
- 能力/风险：雷达或水平条形图（渐变填充）；无需额外库可用原生 SVG。
- 时间轴：用于事件/履历/绯色履历，节点带时间与摘要，可展开详情。
- 计分卡：数字 + 迷你条/进度，颜色按语义。
- 派系/关系：简化矩阵或小型力导（节点≤12时使用），提供文字备份视图以防图形受限。
- 关系网图例：左下图例栏展示各关系类型的颜色/线型；右上显示节点计数与过滤状态。
- 全量变量页：分区卡片右上显示“已填/总字段”计数；分页区显示“第 n/总页”；对 record/数组显示行数。

## 9. 可访问性与键盘
- 键盘：Tab 顺序、Enter 提交、Esc 关闭抽屉；Ctrl/Cmd+S 触发表单保存。
- 对比度：文本与背景对比度 ≥ 4.5；禁用态降低饱和度。
- Icon：FontAwesome 免费图标；为操作按钮添加 `aria-label`。

## 10. 隐私与安全
- 绯色/把柄/人情债默认模糊；导出数据需确认；敏感写入提示“不可撤回”。
- 日志：避免在 console 打印敏感字段内容，使用摘要/ID。

## 11. 性能与降级
- 长列表分页或虚拟列表；抽屉懒加载详情。
- 图形无法渲染时显示文本备选。
- 低带宽模式：可关闭渐变/动效，退化为平面色。
