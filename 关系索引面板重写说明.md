# 关系索引面板重写说明

## 修改概述

已成功重写 `src/绯色官途前端界面/views/Dashboard.vue` 中的"关系索引"面板，实现了动态显示所有字段的功能。

## 主要改进

### 1. **动态变量显示** ✅
- 使用 `computed` 属性动态获取 `关系索引` 对象下的所有字段
- 自动识别字段类型（单值字符串 vs 数组）
- 实时响应变量变化，当LLM通过json patch添加新字段时自动显示

### 2. **移除图标** ✅
- 完全移除了所有字段前的图标（如皇冠、盾牌、心形等）
- 保持界面简洁统一

### 3. **多彩标签** ✅
- 定义了15种不同颜色的颜色池
- 为每个字段（包括初始字段和动态新增字段）分配不同颜色
- 使用CSS变量 `--field-color` 动态应用颜色

### 4. **标签文本处理** ✅
- 使用正则表达式 `/(列表|集合|清单|名单|组|队|群)$/g` 移除字段名末尾的后缀
- 支持多种可能的后缀词：列表、集合、清单、名单、组、队、群
- 示例：
  - "核心嫡系列表" → "核心嫡系"
  - "竞争对手集合" → "竞争对手"
  - "靠山名单" → "靠山"

### 5. **组件复用** ✅
- 继续使用现有的 `CharacterName` 组件显示角色信息
- 保持与整体UI风格的一致性

## 技术实现

### 颜色池定义
```typescript
const FIELD_COLORS = [
  '#d8a657', // 金色
  '#9b59b6', // 紫色
  '#00bcd4', // 天蓝色
  '#e67e22', // 橙金色
  '#ff4d6d', // 绯色
  '#1abc9c', // 青色
  '#ff6b6b', // 红色
  '#4ac18e', // 绿色
  '#f39c12', // 橙色
  '#3498db', // 蓝色
  '#e91e63', // 粉色
  '#8e44ad', // 深紫色
  '#16a085', // 深青色
  '#c0392b', // 深红色
  '#27ae60', // 深绿色
];
```

### 字段名处理函数
```typescript
function processFieldName(fieldName: string): string {
  return fieldName.replace(/(列表|集合|清单|名单|组|队|群)$/g, '');
}
```

### 动态字段计算
```typescript
const 动态关系字段 = computed(() => {
  const fields = [];
  const relationData = 关系索引.value;
  
  let colorIndex = 0;
  for (const [key, value] of Object.entries(relationData)) {
    if (value === null || value === undefined) continue;
    
    const isSingle = typeof value === 'string';
    const displayName = processFieldName(key);
    const color = FIELD_COLORS[colorIndex % FIELD_COLORS.length];
    
    fields.push({ key, displayName, value, isSingle, color });
    colorIndex++;
  }
  
  return fields;
});
```

## 模板结构

```vue
<div class="relation-row">
  <template v-for="field in 动态关系字段" :key="field.key">
    <!-- 单值字段 -->
    <div v-if="field.isSingle && field.value !== '无'" 
         class="relation-chip" 
         :style="{ '--field-color': field.color }">
      <span class="rel-label">{{ field.displayName }}</span>
      <CharacterName :name="field.value" />
    </div>
    
    <!-- 数组字段 -->
    <div v-else-if="!field.isSingle && field.value.length" 
         class="relation-group" 
         :style="{ '--field-color': field.color }">
      <span class="group-label">{{ field.displayName }}</span>
      <div class="group-tags">
        <CharacterName v-for="name in field.value" :key="name" :name="name" />
      </div>
    </div>
  </template>
</div>
```

## 样式优化

- 移除了所有硬编码的颜色类（`.boss`, `.superior`, `.patron` 等）
- 使用CSS变量 `var(--field-color)` 实现动态颜色
- 保持了原有的布局和间距设计
- 继续支持响应式布局和滚动

## 功能特性

✅ **实时响应**：使用Vue的响应式系统，数据变化时自动更新UI
✅ **未知字段支持**：能够正确处理任何未知字段名
✅ **空值过滤**：自动过滤 `null`、`undefined` 和 `'无'` 值
✅ **数组判断**：自动识别字符串和数组类型
✅ **颜色循环**：当字段数超过15个时，颜色会循环使用
✅ **UI一致性**：保持与Dashboard其他部分的视觉风格一致
